🔒 Privacy Policy - DuoMind

Effective Date: January 5, 2025

📱 Application Permissions

DuoMind requests the following permissions for proper functionality:

📷 Camera (android.permission.CAMERA)

  • Purpose: Select or take photos for user profile pictures
  • Usage: Only when user decides to change their profile photo
  • Storage: Images are NOT stored on our servers
  • Optional: You can use the app without granting this permission
  • Control: You can revoke this permission anytime in device settings

🔐 Data Collection and Use

  • We only collect data necessary for app functionality
  • Chat conversations are stored securely and encrypted
  • We do not share personal information with third parties
  • We use Firebase (Google) as our secure backend infrastructure
  • AI analysis is performed to provide insights

🛡️ Data Security

  • We implement industry-standard security measures
  • Data is transmitted through encrypted connections (HTTPS/TLS)
  • Only authenticated users can access their data
  • We follow Google Firebase security best practices

👫 User Connectivity

  • Data is shared only between connected user accounts
  • Each user controls their own information
  • Users can delete their data at any time
  • User linking requires mutual consent via invite codes

🤖 AI and Analytics

  • We use AI to provide insights and session analysis
  • All AI processing respects user privacy
  • No personal data is used to train external AI models
  • Analytics are used to improve app functionality

💳 Premium Subscriptions

  • Payment processing is handled securely by Google Play
  • We do not store payment information
  • Subscription data is used only for service provision

🔄 Policy Updates

This policy may be updated occasionally. We will notify users of significant changes through the app or email.

🌍 International Users

This app is developed in Colombia and follows international privacy standards including GDPR compliance for European users.
